unmuzzle

Censorship-resistant distribution for open-weight models. Built for AI agents, usable by humans.

pip install unmuzzle

Why

No gates, no auth, no click-throughs

Hugging Face gated models need a human in a browser. Here the whole flow is scriptable: plain-JSON index, --json on every command, an MCP server.

Censorship-resistant by construction

The index is plain JSON in a git repo anyone can mirror. Weights move over web-seeded BitTorrent and HTTP mirrors, so no central server can be blocked or billed.

Signed, not trusted

Every file is sha256-pinned and every manifest is minisign-signed. The signature is the trust root, not the host.

Zero-friction install

Downloads land in the Hugging Face cache layout, so from_pretrained(..., local_files_only=True) just works.

Install

pip install unmuzzle
brew install aria2 minisign   # optional: torrents, signature verification

Registry

1 model in the index. Every entry is signed; the CLI verifies the signature and every sha256 before install.

Publish

Any model with a redistribution-friendly license can join the index. The full protocol (signing, mirrors, web-seeded torrents, verification) is one file: AGENTS.md. It is written for AI agents, and it is exactly what a human would do by hand.

Trust root

Releases are signed with minisign. Publisher public key:

RWQGj//qSw5zlf6EB/sSfp8thdSO5ZBr+5pAC9bGhjjReQZ+PLs4gF4K

If a mirror serves bytes that fail sha256 against a signed manifest, the mirror is wrong, not the index. The CLI also pins publisher keys on first use (SSH-style TOFU) and aborts if one ever changes; the official unmuzzle key is pinned inside the pip package itself, so no index host alone can substitute it.
